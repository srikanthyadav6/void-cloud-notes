# RecallStack

Static personal interview-prep dashboard.

The goal is not to build a normal notes app. The goal is to help answer one question:

```text
Can I explain this in an interview without looking?
```

## Direction

RecallStack is built around:

- Topic cards
- Standard interview-note templates
- Recollection mode
- Common traps and follow-up questions
- Dense cheat sheets
- Interviewer expectation insights

The first version is intentionally static. No login, backend, database, admin panel, AI, or cloud sync.

## Stack

- React
- TypeScript
- Vite
- PWA setup
- GitHub Pages deployment

## Pages

## UX


Mobile layout:

```text

Category/menu buttons
Main content
```


Build:
